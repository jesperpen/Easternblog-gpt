# The Eastern Blog

Static site for documenting communist and Soviet-era mosaics, monuments and architecture across Eastern Europe and beyond.

- Frontend: pure HTML + CSS + vanilla JS (no SPA).
- Global styles: `assets/css/main.css`.
- Scripts: `assets/js/main.js`, `assets/js/lightbox.js`, `assets/js/filters.js`.
- Admin HTML views in `/admin`.

You can deploy this repo directly on Netlify, Vercel or any static host.

For AI assistants (Codex): also see `docs/PROJECT_GUIDE.md`.
